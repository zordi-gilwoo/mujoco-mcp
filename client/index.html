RL Environment Creation

Structure your RL environment prompt with:

  • Task Type: reaching, balancing, walking, manipulation
  • Robot Type: franka_panda, cart_pole, quadruped, simple_arm
  • Observation Space: joint positions, velocities, end-effector pose
  • Action Space: joint torques, position commands, velocity commands
  • Reward Function: distance to target, stability, efficiency
  • Episode Termination: task completion, time limit, failure conditions

Example Prompts:

  • "Create a reaching task for Franka Panda to reach random targets"
  • "Design a cart-pole balancing environment with discrete actions"
  • "Build a quadruped walking environment with gait rewards"

Note: The RL environment will use the scene created above as its base. Make sure to generate a scene first!
